# Dario Didaj — Interactive CV

> Where imagination meets artificial intelligence.

A cinematic, fantasy-inspired interactive CV / personal portfolio built for **Dario Didaj** — Digital Marketing student, AI Director and cinematic storyteller behind the [Idrako](https://www.youtube.com/@Idrako.official) project.

🌐 **Live site:** _add your GitHub Pages URL here once deployed_

---

## ✨ Features

- 🐉 **Cinematic dark/fantasy design** with the Idrako dragon as hero centerpiece
- 🌍 **Bilingual switch** (Italian / English) with localStorage persistence
- 🎬 **YouTube embeds** for Idrako music videos (Latest releases)
- 📊 **Animated skill bars** for AI tools and Graphic Design stack
- ✨ **Particle background** with custom canvas animation
- 🖱️ **Custom cursor** with glow effect (desktop only)
- 📱 **Fully responsive** (desktop, tablet, mobile)
- ⚡ **Zero dependencies** — pure HTML, CSS, JS (only Google Fonts via CDN)
- 🎨 **Custom typography** (Cinzel display + Inter body + JetBrains Mono UI)

## 🚀 Quick start (local)

Just open `index.html` in a browser, or serve it locally:

```bash
# Python 3
python3 -m http.server 8080

# Node (if you have http-server)
npx http-server -p 8080
```

Then visit `http://localhost:8080`.

---

## 🌐 Deploy on GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Under **Source**, select branch `main` and folder `/ (root)`
4. Click **Save**
5. Wait ~1 minute and visit `https://<your-username>.github.io/<repo-name>/`

Optional: connect a custom domain (e.g. `dariodidaj.com`) from the Pages settings.

## 🎨 Tech & Design

- **Stack:** Vanilla HTML / CSS / JavaScript (no build step, no framework)
- **Fonts:** Cinzel (display) · Inter (body) · JetBrains Mono (UI)
- **Color system:** Glacial blue palette (`#02040a` → `#4cc8ff`)
- **Animations:** CSS transitions + canvas particles + intersection observer reveals
